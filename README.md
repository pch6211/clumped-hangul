+ 뭉친 한글 (Clumped Hangul) v1.0

뭉친 한글은 한글의 모아쓰기를 물리적으로 구현한 시뮬레이션 공간입니다.
한 글자는 하나의 중심을 가지고 있고, 자음과 모음을 중앙으로 끌어당깁니다.
중앙에서 만난 획들은 서로 밀거나 파고들며 그 글자만의 균형을 찾아갑니다.

+ 주요 기능

1. 물리 기반 타이포그래피 제어 (설정 창 - S)
   - 실시간 인터랙션: 마우스 드래그를 통해 글자의 점(Node) 위치를 직접 이동시키며 실시간으로 모양을 흩트리거나 변형할 수 있습니다.
   - 디테일한 속성 조절: 설정 패널을 통해 텍스트 내용, 글자 크기, 색상뿐만 아니라 자간(Alt + ⇄)과 행간(Alt + ⇅)까지 제어할 수 있습니다.
   - 랜덤 및 자동화: 주사위(⚀) 아이콘이나 무작위 변경(R) 키를 눌러 형태를 탐험할 수 있습니다.
   - 5초 주기의 자동 무작위 전환(A) 모드도 지원합니다.

2. 나만의 자모 디자인 (편집 창 - E)
   - 직접 그리는 글자: 마우스 좌클릭으로 점과 획을 생성하고, 선택 항목을 지우며(Delete) 자모를 디자인할 수 있습니다.
   - 편집 도구: 다중 선택, 방향키를 이용한 정밀 이동, 우클릭을 통한 복사/잘라내기/붙여넣기(Ctrl + C/X/V) 기능을 제공합니다.
   - 골격 세분화: 초성, 중성, 종성 각각의 너비와 높이를 6개의 슬라이더로 조절하여 비례를 맞출 수 있습니다.

3. 물리 효과 시스템
   - 투명 노드: 화면에는 보이지 않지만 획을 밀어내거나 끌어당기는 물리적 기능은 유지하는 속성을 추가할 수 있습니다.
   - 자석 노드: '앞에 붙임' 또는 '뒤에 붙임' 속성을 부여하여 특정 자모에 물리적으로 반응하도록 설계할 수 있습니다.

4. 문자 관리 및 내보내기
   - 확장성: 기본 한글 자모 외에도 유니코드 블록 단위나 개별 코드 입력을 통해 새로운 문자를 추가할 수 있습니다.
   - 저장 및 복원(SAVE/LOAD): 제작한 자모 디자인과 설정값을 JSON 파일로 백업하고 복원할 수 있습니다.
   - 고품질 렌더링: 시뮬레이션 화면을 벡터 이미지(SVG, Ctrl + S) 또는 비트맵(PNG, Ctrl + P)으로 저장할 수 있습니다.
