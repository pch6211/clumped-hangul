++ 뭉친 한글 (Clumped Hangul) v1.10

뭉친 한글은 한글의 모아쓰기를 물리적으로 구현한 시뮬레이션 공간입니다. 한 글자는 하나의 중심을 가지고 있고, 자음과 모음을 중앙으로 끌어당깁니다. 획들은 중앙에서 만나 서로 밀거나 파고들며 그 글자만의 균형을 찾아갑니다.

+ v1.10에 새롭게 추가된 것들

쌍자음(ㄲ·ㄸ·ㅃ·ㅆ·ㅉ), 겹받침(ㄺ·ㄵ·ㄶ 등), 합성모음(ㅒ·ㅔ·ㅘ·ㅙ 등)을 단독으로 입력해도 그 안의 자모들끼리 본드로 붙일 수 있게 됨
획 하나에 다른 두 개 이상의 획을 한꺼번에 붙일 수 있게 됨
본드로 붙어 있는 접합점을 드래그하면 떨어지거나 새로 붙지 않고 두 자모가 함께 끌려옴
한 글자에서 만든 본드가 같은 자모를 가진 다른 모든 글자에 자동 적용 — '낄'의 ㄲ을 붙이면 '끾'의 받침 ㄲ과 단독 ㄲ에도 그대로 적용됨
관리자 모드에 '본드 위치 무관' 토글 추가 (기본 켜짐) — 다른 자모 사이의 본드(예: ㅇ↔ㅓ)도 모임꼴 자리에 상관없이 적용. '어'에서 ㅇ↔ㅓ를 붙이면 '워·얼·월'의 ㅇ↔ㅓ도 함께 붙음
화면의 글자를 우클릭하면 그 글자의 편집 창으로 바로 진입
본드를 떼는 데 필요한 끌기 거리 기본값을 0으로 — 살짝만 끌어도 즉시 분리
임의 텍스트(T 키)에서 한 글자 단어(꿈·꿩 등)는 8번 중 1번, 새 단어 풀은 5번 중 1번 꼴로 등장하도록 비율 조정. 새 문장도 풀에 추가
관리자 모드 슬라이더 기본값 일괄 재조정 (자간 보정·화면 채움 임계·분할 감소 속도·떼기 임계 등 8개)

+ 출력 화면에 대해서

텍스트 입력창에 입력한 글자가 화면에 등장. 한글·영문·숫자·기호·이모지 모두 사용 가능
출력된 글자의 획을 마우스로 끌어 자유롭게 움직임. 놓는 순간 그 자리가 새 안정점이 됨
같은 글자 안의 다른 자모 획 끝(기본 노드) 위로 끌어다 놓으면 그 두 노드가 본드로 결합. 이후 항상 붙어서 출력됨
본드로 붙어 있는 노드를 한 번 더 끌면 떨어짐
본드의 접합점에 마우스를 올리면 양쪽 자모가 함께 강조 표시됨. 접합점을 드래그하면 결합 상태 그대로 같이 끌려옴
글자를 우클릭하면 그 글자의 편집 창이 곧바로 열림
빈 공간을 좌클릭 드래그하면 가로·세로 픽셀 거리 측정자가 그려짐
A 키 — 자동 재생 (5초마다 임의 텍스트 + 임의 설정 값 갱신)
T 키 — 임의 텍스트로 교체
R 키 — 임의 설정 값으로 슬라이더 일괄 변경
B 키 — 글자 리셋 (위치·각도 초기화)
M 키 — 가이드(단축키 안내 HUD) 표시 토글
Alt + ←/→ — 자간 조정 (±5)
Alt + ↑/↓ — 행간 조정 (±10)
Ctrl + Z / Ctrl + Shift + Z — 본드 결합·해제 실행취소·다시실행
Ctrl + S — 현재 화면을 SVG로 저장
Ctrl + P — 현재 화면을 PNG로 저장

+ 설정 창에 대해서 (S 키)

점간격, 점 크기, 획 크기 슬라이더로 글자의 기본 골격 조정
자간·행간 슬라이더로 글자 사이·줄 사이 간격 조정
골격 분할(가로/세로) 슬라이더로 획에 노드 추가 분할
중력·반발·모양 복원력·감쇠·속도 슬라이더로 시뮬레이션 물리 조정
자석 힘·자석 반경 슬라이더로 본드 결합의 강도·범위 조정
색상 선택기로 배경색·점 색·획 색·강조 색 변경
꾸밈 그룹의 '테두리'·'부드럽게' 등 시각효과 토글
자모 JSON 파일 불러오기·저장하기로 디자인 일괄 교체·보관
'전체 초기화' 버튼으로 localStorage 비우고 공장 기본값으로 복원
드래그 핸들을 1.5초 안에 10번 빠르게 클릭하면 관리자 모드 다이얼로그가 열림
관리자 모드에서는 화면 채움 임계·축소 비율·자간/행간 보정 계수·분할 곡선 등 내부 변수들의 원시 값 직접 입력
관리자 모드 '본드 위치 무관' 항목 (0/1) — 1이면 본드가 모임꼴 자리에 상관없이 같은 글자 모든 위치에 자동 적용
관리자 모드의 '기본값 저장'으로 현재 값을 localStorage에 영속 저장. ↺ 버튼으로 각 항목 공장값 복원. '코드 복사'로 변경된 값들의 JS 스니펫을 클립보드로 출력

+ 편집 창에 대해서 (E 키, 또는 출력 글자 우클릭)

사이드바의 자모 리스트에서 편집할 글자 선택. 텍스트 입력창에 입력한 문자·기호·이모지가 이 리스트에 자동 추가됨
캔버스 빈 공간을 클릭하여 새 노드 추가
노드를 클릭·드래그하여 위치 이동. 격자(0.5 단위)에 자동 스냅
두 노드를 차례로 클릭하면 그 사이에 획 추가
획 위를 클릭하면 그 획 선택. Delete/Backspace 키로 선택된 노드·획 삭제
획을 더블클릭하면 '투명 획' 토글 — 노드는 그대로 두고 시각 렌더링만 끔
화살표 키 — 선택한 노드(들)를 0.5 단위로 이동
Ctrl + C / X / V — 선택한 노드 묶음 복사·잘라내기·붙여넣기
Ctrl + Z / Ctrl + Shift + Z — 편집 동작 실행취소·다시실행
'초기화' 버튼 — 현재 자모를 공장 기본값(혹은 마지막 로드된 JSON)으로 복원
'모임꼴' 토글 — 박스 배치 모드 진입. 한 음절 안에서 자모들이 차지할 박스 위치를 자유롭게 조정. 박스 6 패턴(가로/세로/섞임 × 받침 유무) 각각 따로 저장
모임꼴 모드에서 박스를 드래그하여 위치 변경. 화살표 키로 1 칸씩 미세 이동. 박스 크기를 키우면 그 자모의 획 분할이 추가됨
자모 본드(앞/뒤 결합)는 출력 화면 드래그로만 만들고 편집 창에서는 결과만 표시됨. 본드된 노드에는 's'·'n' 태그가 자동 부여되어 보임
