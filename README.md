뭉친 한글 (Clumped Hangul) v1.09
  - 뭉친 한글은 한글의 모아쓰기를 물리적으로 구현한 시뮬레이션 공간입니다. 한 글자는 하나의 중심을 가지고 있고, 자음과 모음을 중앙으로 끌어당깁니다. 획들은 중앙에서 만나 서로 밀거나 파고들며 그 글자만의 균형을 찾아갑니다.

v1.09에 새롭게 추가된 것들
  - 출력 화면에서 자모를 우클릭하면 메뉴 표시 (자모 편집, 모임꼴 편집, 획 붙임 해제)
  - 디자인이 없는 새 문자를 입력하면 ㅁ 안에 ✕ 모양으로 자동 출력. 자모 선택창에도 자동 추가
  - PC 텍스트 입력창을 출력 화면 하단 중앙으로 분리. 설정창을 닫아도 항상 보임
  - 기본 HUD를 화면 왼쪽 위로 이동. 다른 가이드와 같은 박스 디자인 적용. 항목별로 행갈이
  - 자모 디자인창의 8 변형 핸들 삭제. 우클릭 메뉴에 좌우 반전, 상하 반전 추가
  - 투명획에만 연결된 노드는 자동으로 투명 노드로 처리
  - 후원창 QR 코드 표시 1초 지연 제거. 로컬에서 즉시 생성

v1.08에 새롭게 추가된 것들
  - 출력 화면의 획을 끌어당겨 같은 글자의 노드에 붙이면 그 조합이 항상 붙어서 출력되는 기능 신설
  - 편집 창의 앞/뒤에 붙임 기능 삭제
  - 텍스트 입력창에 입력한 문자/기호/이모지가 편집 창의 문자 리스트에 자동으로 추가
  - 설정 창의 꾸밈 그룹에 시각효과 '테두리' 신설
  - 자음에 쌍자음, 모음에 합성모음을 추가하여 모임꼴 종류를 30종에서 6종으로 단순화

출력 화면에 대해서
  - 입력한 글자들이 화면에 떠다님. 마우스로 획을 잡고 끌어당기면 그 자리에 고정
  - 글자 위에 마우스를 올리면 디자인의 기본 점만 강조 표시
  - 강조된 점을 잡고 같은 글자의 다른 자모 점으로 끌어가면 자석처럼 붙음 (= 붙임)
  - 편집창에서 손으로 추가한 점을 잡고 끌면 글자에서 떼어짐 (= 떼기)
  - 글자를 우클릭하면 메뉴 표시 — 자모 편집, 모임꼴 편집, 획 붙임 해제
  - 화면 왼쪽 위에 단축키 안내 HUD 박스. M 키로 끄고 켜기
  - 클릭으로 실행되는 단축키 항목은 점선 밑줄로 표시
  - 화면 빈 영역을 마우스로 드래그하면 수직 또는 수평으로 픽셀 거리 측정 (개발용)

설정 창에 대해서
  - S 키로 열고 닫음. 모바일은 좌상단 햄버거 ☰ 버튼
  - 첫 줄 텍스트 입력창은 출력 화면 하단 중앙에 따로 표시 — 설정창을 닫아도 보임
  - 텍스트 입력창은 줄이 늘어남에 따라 자동으로 길어짐. 크기 조절 핸들 없음
  - 그 아래 슬라이더 그룹들. 각 그룹 제목 옆에 주사위 ⚄ 클릭 = 그 그룹만 랜덤
  - 슬라이더 옆 숫자에 점선 밑줄 — 클릭 시 직접 값 입력 가능
  - 입력바의 주사위 = 글귀 랜덤 (T 단축키와 동일)
  - 모바일 패널은 좌측 3/4 차지. 우측 1/4 영역을 탭하면 자동 닫힘
  - 모바일 패널 우상단 ✕ 버튼으로도 닫기. 다른 팝업 닫기 버튼과 동일 디자인

편집 창에 대해서
  - E 키로 열고 닫음
  - 왼쪽 사이드바 — 자모 선택창 (한글 자음, 모음, 라틴, 기호, 이모지 등 카테고리별 버튼)
  - 카테고리 제목 옆 ✕ 버튼 = 카테고리 닫기
  - 자모 버튼 우클릭 = 그 문자 삭제
  - 선택한 자모의 디자인이 오른쪽 스테이지에 표시. 점 (노드), 선 (획) 추가, 이동, 삭제 가능
  - 펜으로 빈 그리드 점이나 기존 획 위 클릭 = 새 노드 추가
  - 두 노드 사이 드래그 = 새 획
  - 우클릭 메뉴 — 삭제, 복사, 잘라내기, 붙여넣기, 투명하게/불투명하게, 좌우 반전, 상하 반전
  - 다중 선택 후 선택 박스 안에서 우클릭 = 선택 유지, 메뉴만 표시
  - 모임꼴 6 종 (미, 므, 믜, 밈, 믐, 믬) 도 동일한 방식으로 편집
  - 출력 화면에서 자모 우클릭 → 자모 편집/모임꼴 편집 시 사이드바는 숨기고 스테이지만 표시
  - 사이드바 숨김 시 스테이지 기본 위치는 화면 우측 끝. 드래그로 이동 가능 (위치 저장됨)
