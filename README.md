# 뭉친 한글 (Clumped Hangul)

> 한글의 모아쓰기를 물리적으로 구현한 시뮬레이션 공간.

한 글자는 하나의 중심을 가지고 있고, 자음과 모음을 중앙으로 끌어당깁니다. 획들은 중앙에서 만나 서로 밀거나 파고들며 그 글자만의 균형을 찾아갑니다.

**버전**: v1.11 (2026-05-23)

---

## v1.11 변경사항

- ㅅ↔ㅅ 결합 교점에 ㅗ를 새로 붙일 수 있게 됨
- 자모별 골격 슬라이더가 6개로 늘어남
- 임의 획 추가 분배가 자연스러워짐
- 글자 수가 많을수록 분할 슬라이더로 추가되는 노드 수가 자동으로 줄어듦
- 합성중성 ㅐㅒㅔㅖ가 쌍자음처럼 처리됨
- 편집창에서 새로 그린 노드도 호버·자석 본드 후보로 즉시 잡힘
- 획 중간 노드를 삭제해도 양 이웃이 자동으로 이어져 획이 유지됨
- 이어그리기 중 `Del` 키로 그리던 노드 바로 삭제
- 노드 우클릭 메뉴에 **투명하게 / 불투명하게** 추가
- 노드는 자동으로 투명해지지 않음
- 모바일 전용 UI 폐기
- 모바일에서 길게 탭(0.5초)이 우클릭으로 동작
- 모바일에서 우클릭 메뉴가 떴을 때 항목이 미리 선택되어 있던 현상 해소.
- 가이드 안 단어가 환경에 따라 자동 변환
- 의견 보내기가 메일 클라이언트 없이도 동작
- 임의 텍스트(`T` 키)의 단글자·단어·문장 각 풀이 한 사이클 동안 같은 항목을 중복하지 않게 됨

## v1.10 변경사항

- 쌍자음(ㄲ·ㄸ·ㅃ·ㅆ·ㅉ), 겹받침(ㄺ·ㄵ·ㄶ 등), 합성모음(ㅒ·ㅔ·ㅘ·ㅙ 등)을 단독으로 입력해도 그 안의 자모들끼리 본드로 붙일 수 있게 됨
- 획 하나에 다른 두 개 이상의 획을 한꺼번에 붙일 수 있게 됨
- 본드로 붙어 있는 접합점을 드래그하면 떨어지거나 새로 붙지 않고 두 자모가 함께 끌려옴
- 한 글자에서 만든 본드가 같은 자모를 가진 다른 모든 글자에 자동 적용
- 관리자 모드에 **본드 위치 무관** 토글 추가 (기본 켜짐)
- 화면의 글자를 우클릭하면 그 글자의 편집 창으로 바로 진입
- 본드를 떼는 데 필요한 끌기 거리 기본값을 0으로
- 임의 텍스트 풀 비율 조정
- 관리자 모드 슬라이더 기본값 일괄 재조정

## 출력 화면

### 텍스트 입력

- 텍스트 입력창에 입력한 글자가 화면에 등장. 한글·영문·숫자·기호·이모지 모두 사용 가능
- 입력한 문자는 편집 창의 자모 리스트에도 자동 추가됨

### 노드 조작

- 출력된 글자의 획을 마우스로 끌어 자유롭게 움직임. 놓는 순간 그 자리가 새 안정점이 됨
- 빈 공간을 좌클릭 드래그하면 가로·세로 픽셀 거리 측정자가 그려짐

### 본드 (획 잇기)

- 같은 글자 안의 다른 자모 획 끝(기본 노드) 위로 끌어다 놓으면 그 두 노드가 본드로 결합. 이후 항상 붙어서 출력됨
- 본드로 붙어 있는 노드를 한 번 더 끌면 떨어짐
- 본드의 접합점에 마우스를 올리면 양쪽 자모가 함께 강조 표시됨. 접합점을 드래그하면 결합 상태 그대로 같이 끌려옴
- 결합 교점 위에서 다른 자모로 새 본드 시작 가능 (떼기는 단순 드래그만)

### 편집 진입

- 글자를 우클릭하면 그 글자의 편집 창이 곧바로 열림
- 터치 디바이스에서는 길게 탭 (500ms)으로 우클릭

### 단축키

| 키 | 기능 |
|---|---|
| `A` | 자동 재생 (5초마다 임의 텍스트 + 임의 설정 값 갱신) |
| `T` | 임의 텍스트로 교체 (카테고리 한 사이클 동안 중복 없음) |
| `R` | 임의 설정 값으로 슬라이더 일괄 변경 |
| `B` | 글자 리셋 (위치·각도 초기화) |
| `M` | 가이드(단축키 안내 HUD) 표시 토글 |
| `Alt` + `←` / `→` | 자간 조정 (±5) |
| `Alt` + `↑` / `↓` | 행간 조정 (±10) |
| `Ctrl` + `Z` | 본드 결합·해제 실행취소 |
| `Ctrl` + `Shift` + `Z` | 본드 결합·해제 다시실행 |
| `Ctrl` + `S` | 현재 화면을 SVG로 저장 |
| `Ctrl` + `P` | 현재 화면을 PNG로 저장 |

## 설정 창 (`S` 키)

### 기본 슬라이더

- **점 간격 · 점 크기 · 획 크기** — 글자의 기본 골격 조정
- **자간 · 행간** — 글자 사이·줄 사이 간격 조정
- **골격 분할** — 자모별 가로/세로 노드 분할 (초성 너비/높이, 중성 너비/높이, 종성 너비/높이)
- **임의 획 추가** — 가장 긴 획부터 자연스럽게 분할
- **중력 · 반발 · 모양 복원력 · 감쇠 · 속도** — 시뮬레이션 물리 조정
- **자석 힘 · 자석 반경** — 본드 결합의 강도·범위 조정

### 꾸밈

- 색상 선택기로 배경색·점 색·획 색·강조 색 변경
- `테두리` · `부드럽게` 등 시각효과 토글

### 파일 입출력

- 자모 JSON 파일 불러오기·저장하기로 디자인 일괄 교체·보관
- `전체 초기화` 버튼으로 localStorage 비우고 공장 기본값으로 복원

### 관리자 모드

> 드래그 핸들을 1.5초 안에 10번 빠르게 클릭하면 다이얼로그가 열립니다.

- 화면 채움 임계·축소 비율·자간/행간 보정 계수·분할 곡선 등 내부 변수들의 원시 값 직접 입력
- **본드 위치 무관** 항목 (0/1) — 1이면 본드가 모임꼴 자리에 상관없이 같은 글자 모든 위치에 자동 적용
- **분할 제한 토글** — ON: 글자수에 따라 슬라이더 max·동시 활성 개수 제한 / OFF: 모든 분할 0~5 자유
- `기본값 저장` — 현재 값을 localStorage에 영속 저장
- ↺ — 각 항목 공장값 복원
- `코드 복사` — 변경된 값들의 JS 스니펫을 클립보드로 출력

## 편집 창 (`E` 키, 또는 출력 글자 우클릭)

### 자모 선택

- 사이드바의 자모 리스트에서 편집할 글자 선택
- 텍스트 입력창에 입력한 문자·기호·이모지가 이 리스트에 자동 추가됨

### 노드·획 편집

- 캔버스 빈 공간을 클릭하여 새 노드 추가 (즉시 호버·자석 본드 후보로 잡힘)
- 노드를 클릭·드래그하여 위치 이동. 격자(0.5 단위)에 자동 스냅
- 두 노드를 차례로 클릭하면 그 사이에 획 추가
- 획 위를 클릭하면 그 획 선택
- 획 중간의 노드를 삭제하면 양 이웃이 자동으로 연결되어 획이 유지됨
- 이어그리기 중 `Del` 키로 그리던 노드 바로 삭제

### 투명 획 / 투명 노드 (우클릭 메뉴)

- 노드 또는 획을 선택한 후 우클릭 → **투명하게 / 불투명하게** 토글
- 다중 선택 (노드+획) 시 둘 다 일괄 토글
- 투명 처리해도 물리(밀어내기·자석)는 그대로 작동 — 시각 렌더링만 끔
- 노드는 자동으로 투명해지지 않음 — 사용자가 명시적으로 토글한 경우만 inv

### 본드 표시

- 자모 본드(앞/뒤 결합)는 출력 화면 드래그로만 만들고 편집 창에서는 결과만 표시
- 본드된 노드에는 `s` · `n` 태그가 자동 부여되어 보임

### 초기화

- `초기화` 버튼 — 현재 자모를 공장 기본값(혹은 마지막 로드된 JSON)으로 복원

### 모임꼴 모드

- 박스 배치 모드 진입. 한 음절 안에서 자모들이 차지할 박스 위치를 자유롭게 조정
- 박스 6 패턴(가로 / 세로 / 섞임 × 받침 유무) 각각 따로 저장
- 박스를 드래그하여 위치 변경
- 박스 크기를 키우면 그 자모의 획 분할이 추가됨
- 쌍초성(cho1↔cho2), 합성중성(jung1↔jung2), 쌍/겹받침(jong1↔jong2)은 한 박스가 대표 — 한쪽만 정의하면 다른 쪽 자동 미러

### 단축키

| 키 | 기능 |
|---|---|
| `←` / `→` / `↑` / `↓` | 선택한 노드(들)를 0.5 단위로 이동 |
| `Delete` / `Backspace` | 선택된 노드·획 삭제 / 이어그리기 중이면 펜 노드 삭제 |
| `Ctrl` + `C` | 선택한 노드 묶음 복사 |
| `Ctrl` + `X` | 선택한 노드 묶음 잘라내기 |
| `Ctrl` + `V` | 노드 묶음 붙여넣기 |
| `Ctrl` + `Z` | 편집 동작 실행취소 |
| `Ctrl` + `Shift` + `Z` | 편집 동작 다시실행 |
| 모임꼴 모드에서 `←` / `→` / `↑` / `↓` | 박스를 1 칸씩 미세 이동 |
