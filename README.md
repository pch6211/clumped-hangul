+ 뭉친 한글 (Clumped Hangul) v1.08

  뭉친 한글은 한글의 모아쓰기를 물리적으로 구현한 시뮬레이션 공간입니다. 한 글자는 하나의 중심을 가지고 있고, 자음과 모음을 중앙으로 끌어당깁니다. 획들은 중앙에서 만나 서로 밀거나 파고들며 그 글자만의 균형을 찾아갑니다.

+ v1.08에 새롭게 추가된 것들
- 출력 화면의 획을 끌어당겨 같은 글자의 노드에 붙이면 그 조합이 항상 붙어서 출력되는 기능 신설(편집 창의 앞/뒤에 붙임 기능 삭제)
- 설정 창의 텍스트 입력창에 입력한 문자/기호/이모지가 편집 창의 문자 리스트에 자동으로 추가
- 설정 창의 꾸밈 그룹에 '테두리' 신설
- 자음에 쌍자음, 모음에 합성모음을 추가하여 모임꼴 종류를 30종에서 6종으로 단순화(섞임받침 추가 예정)
