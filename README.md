+ 뭉친 한글 (Clumped Hangul) v1.0 +

뭉친 한글은 한글의 모아쓰기를 물리적으로 구현한 시뮬레이션 공간입니다.
한 글자는 하나의 중심을 가지고 있고, 자음과 모음을 중앙으로 끌어당깁니다.
중앙에서 만난 획들은 서로 밀거나 파고들며 그 글자만의 균형을 찾아갑니다.

설정 창(S)에서 텍스트를 입력하고 여러 변수를 조절 할 수 있습니다.
마음에 드는 설정값, 이미지, 벡터 파일을 다운 받고 공유 할 수 있습니다.

편집 창(E)에서 문자를 디자인하거나 문자를 추가 할 수 있습니다.
한글 모임꼴의 초·중·종성의 크기와 위치를 편집할 수 있습니다.
문자와 모임꼴 디자인을 저장하거나 불러올 수 있습니다.

+ 주요 기능 +

1. 실시간 인터랙션: 마우스 드래그를 통해 글자의 점(Node) 위치를 직접 이동시키며 실시간으로 모양을 흩트리거나 변형할 수 있습니다.
2. 물리 기반 타이포그래피 제어 (속성 창 - S)
3. 디테일한 속성 조절: 설정 패널을 통해 텍스트 내용, 글자 크기, 색상뿐만 아니라 자간(Alt + ⇄)과 행간(Alt + ⇅)까지 자유롭게 제어할 수 있습니다.
4. 랜덤 및 자동화: 주사위(⚀) 아이콘이나 무작위 변경(R) 키를 눌러 우연이 만들어내는 기하학적 형태를 탐험할 수 있습니다.
5. 5초 주기의 자동 무작위 전환(A) 모드도 지원합니다.
